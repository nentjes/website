📁 Exporteer workshops-data.js

Kopieer de code hieronder en vervang de inhoud van workshops-data.js in je website map. Push daarna naar git om de site te updaten.

  1. Download het bestand of kopieer de code

  3. Commit en push naar git: git add . && git commit -m "workshop toegevoegd" && git push
  4. Cloudflare deployt automatisch binnen ~1 minuut

Handleiding

🗓 Workshop toevoegen

  1. Ga naar het tabblad Workshops
  2. Klik op + Nieuwe workshop
  3. Vul het formulier in en klik Opslaan
  4. Klik onderin op Download bestand
  5. Vervang workshops-data.js in je website map
  6. Push naar git — Cloudflare deployt automatisch

🔐 Wachtwoord wijzigen

Het wachtwoord staat in admin/index.html. Zoek naar ADMIN_PASSWORD en verander de waarde. Push daarna naar git.

🚀 Project toevoegen

Projecten worden beheerd in index.html in de sectie id="projecten". Vraag Claude Code om een nieuw project toe te voegen.

📝 Tekst aanpassen

Alle tekst op de website staat in de HTML bestanden. Vraag Claude Code om specifieke tekst aan te passen, of edit de bestanden direct in VS Code of een teksteditor.
